HTML

HTML (HyperText Markup Language) is a standard language used to create the layout and structure of web pages. It is not a programming language, but rather a set of tags (skeleton) that tells the browser how to display text, images, and other elements. HTML files are saved with the extension .html or .htm.

CSS

CSS (Cascading Style Sheets) is a stylesheet language used for describing the presentation of a document written in HTML or XML. It controls the layout, colors, fonts, and overall visual appearance of web pages, ensuring consistent design and responsive layouts across different devices.

JavaScript

JavaScript is a high-level, interpreted programming language that is primarily used for creating interactive web pages and dynamic content. It enables developers to implement complex features on websites, such as animations, form validation, and real-time updates.

Tailwind CSS

Tailwind CSS is a utility-first CSS framework for rapidly building custom user interfaces. It provides a set of low-level utility classes that can be combined to create unique designs without writing custom CSS.

GitHub

GitHub is a web-based platform for version control and collaboration, allowing developers to host and review code, manage projects, and build software together.

Vue js

Vue js is a progressive JavaScript framework for building user interfaces. It is designed to be incrementally adoptable and provides a gentle learning curve for developers.
